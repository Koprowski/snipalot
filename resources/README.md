# Bundled resources

At package time, the following binaries and models must live here. Whisper is required for the normal Snipalot experience because recording depends on local transcription.

```
resources/
|-- bin/
|   `-- whisper/
|       |-- whisper-cli.exe  <- from https://github.com/ggerganov/whisper.cpp releases
|       `-- *.dll            <- runtime DLLs from the same release zip
|-- models/
|   `-- ggml-base.en.bin     <- ~150MB, from https://huggingface.co/ggerganov/whisper.cpp
`-- icons/
    `-- app.png              <- generated by scripts/make-icon.mjs
```

ffmpeg is supplied by the `ffmpeg-static` npm dependency and lives under `node_modules/` at build time; electron-builder picks it up automatically via `asar` unpacking rules.

These files are listed in `.gitignore` because they are large and/or platform-specific. `npm run fetch-resources` downloads them, and every installer/portable packaging script runs that command plus `npm run assert-resources` before `electron-builder`. If Whisper is missing, packaging should fail rather than ship a recorder that cannot transcribe.
